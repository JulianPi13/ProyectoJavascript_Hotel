# Proyecto Hotel (JavaScript)

Este proyecto es un sitio web estático para un hotel, construido con HTML, CSS y JavaScript. Está diseñado para mostrar habitaciones, permitir reservar, ver reservas y gestionar el perfil, todo con un diseño adaptado a dispositivos móviles.

## ¿Para qué sirve la página?

La página simula una experiencia de hotel online donde el visitante puede:

- Consultar habitaciones disponibles.
- Reservar habitaciones.
- Ver sus reservas.
- Contactar al hotel.
- Acceder o registrarse en el sistema.
- Ver y editar su perfil.
- Navegar por una interfaz responsive para celular.

## Páginas principales

- `index.html`: Página de inicio de sesión / registro.
- `Inicio.html`: Página principal del hotel con bienvenida y secciones destacadas.
- `habitacion1.html`, `habitacion2.html`, `habitacion3.html`, `habitacion4.html`, `habitacion5.html`: Páginas de detalle para cada habitación.
- `reservas.html`: Formulario para crear una nueva reserva.
- `tus_reservas.html`: Página para ver las reservas realizadas por el usuario.
- `contacto.html`: Página de contacto con información de comunicación.
- `perfil.html`: Página de perfil de usuario.
- `panelcontrol.html`: Panel de administración / control del hotel.

## Estructura del proyecto

- `css/`: Hojas de estilo para cada página.
- `Js/`: Scripts de JavaScript que controlan la navegación y funcionalidades.
- `images/`: Imágenes usadas en la página.
- `Json/`: Datos en formato JSON utilizados por las páginas.

## Características

- Diseño responsive para dispositivos móviles.
- Interfaz moderna con botones y formularios estilizados.
- Navegación entre secciones y páginas.
- Formularios de reserva adaptados a pantallas pequeñas.
- Panel de control pensado para gestión interna.

## Cómo usar

1. Abre `index.html` en un navegador.
2. Navega a través de los enlaces del menú.
3. Ve a `reservas.html` para simular una reserva.
4. Revisa `tus_reservas.html` para ver las reservas del usuario.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Notas
